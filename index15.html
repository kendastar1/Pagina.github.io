El objetivo es reducir el tiempo de ejecución en un 34% respecto al tiempo actual con 8 GB de RAM. Primero, calculamos el tiempo de ejecución actual con 8 GB de RAM.

  • Nueva Tiempo = 50 * ( 1 - 0,34 )
